# 巡礼对比图生成器

地址： https://lab.magiconch.com/image-merge/

## 功能
 - [x] 点击上传
 - [x] 拖拽上传
 - [x] 下载图片
 - [x] 分享图片
 - [x] 从 Anitabi.cn 导入截图 (2023-07-05 做了！)
 - [x] 底色选项 (2023-07-24 做了！)
 - [x] 边框选项
 - [x] 横竖自动切换
 - [x] 根据照片辅助修正地标视点
